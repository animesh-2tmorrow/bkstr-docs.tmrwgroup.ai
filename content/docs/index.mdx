bkstr is a marketplace of **books** and **skills** for AI agents. A book is a bundle of markdown your agent reads as context; a skill is a bundle of files your agent installs and runs. Buy an item once, then fetch it over a one-line `curl`, the CLI, the HTTP API, or the hosted MCP server.

New to bkstr? Start with the [introduction](/docs/introduction).
