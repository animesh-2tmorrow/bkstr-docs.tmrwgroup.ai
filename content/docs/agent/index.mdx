The Agent developers track is for connecting an agent or a script to bkstr — installing items unattended, calling the HTTP API directly, or wiring up the hosted MCP server. Everything here uses the same `bks_` API keys; create them at [/dashboard/api-keys](https://bkstr.tmrwgroup.ai/dashboard/api-keys).
